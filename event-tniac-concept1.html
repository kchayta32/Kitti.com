บทคัดย่อ

งานวิจัยนี้นำเสนอวิธีการวินิจฉัยความผิดปกติของลูกปืนจากสัญญาณการสั่นสะเทือน โดยแปลงสัญญาณให้เป็นภาพแทนเชิงเวลา–ความถี่ด้วยการแปลงเวฟเลตแบบต่อเนื่อง (CWT) เพื่อสร้างภาพสเกโลแกรม และจำแนกชนิดความผิดปกติด้วยโครงข่ายประสาทเทียมเชิงลึกแบบกะทัดรัดบนโปรแกรม MATLAB กระบวนการวิจัยประกอบด้วยการแบ่งสัญญาณเป็นช่วงเวลา การทำให้ข้อมูลอยู่ในรูปมาตรฐาน การสร้างภาพสเกโลแกรม และการฝึกสอนแบบจำลอง โดยใช้ชุดข้อมูลจากมหาวิทยาลัยเคสเวสเทิร์น รัฐโอไฮโอ สหรัฐอเมริกา และกำหนดการจำแนกเป็น 4 กลุ่ม ได้แก่ สภาพปกติ ความเสียหายที่วงแหวนด้านใน ความเสียหายที่วงแหวนด้านนอก และความเสียหายที่เม็ดลูกปืน ทั้งนี้ได้เปรียบเทียบกับวิธีฐานที่ใช้ลักษณะเฉพาะเชิงสถิติร่วมกับเครื่องเวกเตอร์สนับสนุน (SVM) และป่าไม้สุ่ม (Random Forest) ผลการทดลองจะรายงานด้วยค่าความถูกต้อง ความเที่ยง ความครอบคลุม ค่าคะแนนเอฟวัน และเมทริกซ์ความสับสน จากการทดลองบน MATLAB เพียงอย่างเดียว เพื่อยืนยันความเหมาะสมต่อการประยุกต์ใช้ในการบำรุงรักษาเชิงพยากรณ์ในงานอุตสาหกรรม

คำสำคัญ: การวินิจฉัยความผิดปกติของลูกปืน สัญญาณการสั่นสะเทือน CWT สเกโลแกรม Lightweight CNN

ABSTRACT

This paper presents a MATLAB-only bearing fault diagnosis approach from vibration signals. Each signal segment is transformed into a time-frequency representation using Continuous Wavelet Transform (CWT) to generate scalogram images, which are classified by a lightweight convolutional neural network (Lightweight CNN). Experiments use the CWRU dataset with four classes (normal, inner race fault, outer race fault, and ball fault) and compare against baselines based on handcrafted statistical features with SVM and Random Forest. Performance is reported using accuracy, precision, recall, F1-score, and confusion matrices, with all experiments conducted in MATLAB.

Keywords: Bearing fault diagnosis Vibration signals CWT Scalogram Lightweight CNN

I. บทนำ

ความผิดปกติของลูกปืนเป็นปัจจัยสำคัญประการหนึ่งที่ก่อให้เกิดการหยุดชะงักของเครื่องจักรกลหมุน ส่งผลให้กระบวนการผลิตสูญเสียประสิทธิภาพและเกิดความเสียหายทางเศรษฐกิจ การพัฒนาวิธีการวินิจฉัยความผิดปกติจากสัญญาณการสั่นสะเทือนจึงมีบทบาทสำคัญต่อการบำรุงรักษาเชิงพยากรณ์ โดยงานวิจัยร่วมสมัยจำนวนมากรายงานว่าแนวทางการเรียนรู้เชิงลึกสามารถเพิ่มความแม่นยำในการจำแนกสภาวะของเครื่องจักรได้ดีกว่าวิธีการเรียนรู้ของเครื่องแบบดั้งเดิม [1]

ชุดข้อมูลของมหาวิทยาลัยเคสเวสเทิร์น จัดเป็นชุดข้อมูลมาตรฐานที่ได้รับความนิยมอย่างแพร่หลายสำหรับการทดสอบและเปรียบเทียบวิธีการวินิจฉัยความผิดปกติของลูกปืน เนื่องจากจัดเตรียมสัญญาณการสั่นสะเทือนในรูปแบบไฟล์ MATLAB และครอบคลุมทั้งสภาวะปกติและความผิดปกติหลายตำแหน่ง [2] อย่างไรก็ดี งานวิจัยเชิงเปรียบเทียบ ชี้ให้เห็นว่าแนวทางการแบ่งชุดข้อมูลแบบสุ่มในระดับส่วนย่อยของสัญญาณ อาจก่อให้เกิด "ข้อมูลรั่ว" กล่าวคือ ข้อมูลบางส่วนที่มีลักษณะใกล้เคียงกันจากการทดลองเดียวกันอาจปรากฏทั้งในชุดฝึกสอนและชุดทดสอบ ส่งผลให้ค่าความแม่นยำที่รายงานสูงเกินความเป็นจริง ดังนั้นจึงควรกำหนดแผนการแบ่งข้อมูลอย่างระมัดระวัง โดยอิงเงื่อนไขการทดลองหรือรอบการทดสอบเพื่อสะท้อนความสามารถในการประยุกต์ใช้จริงของแบบจำลอง [3], [10]

งานวิจัยนี้เสนอวิธีการจำแนกความผิดปกติของลูกปืนจำนวน 4 คลาส ได้แก่ สภาพปกติ ความเสียหายที่วงแหวนด้านใน ความเสียหายที่วงแหวนด้านนอก และความเสียหายที่เม็ดลูกปืน โดยแปลงสัญญาณการสั่นสะเทือนให้เป็นภาพแทนเชิงเวลา–ความถี่ด้วยการแปลงเวฟเลตแบบต่อเนื่อง (CWT) เพื่อสร้าง "ภาพสเกโลแกรม" ซึ่งคำนวณจากค่าสัมบูรณ์ของสัมประสิทธิ์ CWT และใช้แสดงการกระจายพลังงานของสัญญาณตามเวลาและความถี่ [4], [5] จากนั้นฝึกสอนแบบจำลองโครงข่ายประสาทเทียมเชิงลึกแบบกะทัดรัด (CNN) เพื่อเพิ่มประสิทธิภาพในการจำแนก โดยสอดคล้องกับแนวโน้มงานวิจัยร่วมสมัยที่ประยุกต์ใช้การแปลงเชิงเวลา–ความถี่ เช่น CWT หรือสเปกโตรแกรม ร่วมกับโครงข่ายคอนโวลูชันเพื่อยกระดับความแม่นยำในการวินิจฉัยความผิดปกติ [6]–[9]

II. วัตถุประสงค์ของการวิจัย

  • 1) พัฒนา MATLAB-only pipeline สำหรับวินิจฉัยความผิดปกติของลูกปืนจากสัญญาณสั่นสะเทือน โดยใช้ภาพสเกโลแกรมจาก CWT และ Lightweight CNN
  • 2) เปรียบเทียบประสิทธิภาพกับวิธีฐาน (handcrafted features + SVM และ RandomForest) ภายใต้การแบ่งชุดข้อมูลที่ลดความเสี่ยง data leakage
  • 3) ประเมินผลด้วย Accuracy, Precision, Recall, F1-score, Confusion Matrix และเวลา inference ต่อหนึ่งเซกเมนต์บน CPU

III. วิธีดำเนินการวิจัย

III.I ชุดข้อมูลและการเตรียมข้อมูล

ใช้ชุดข้อมูล CWRU โดยเลือกสัญญาณจากตำแหน่ง drive-end ที่ความถี่การสุ่มตัวอย่าง 12 kHz เพื่อความสม่ำเสมอของข้อมูล [2]. กำหนด 4 คลาส ได้แก่ Normal, Inner Race Fault (IR), Outer Race Fault (OR) และ Ball Fault (B) จากนั้นแบ่งสัญญาณเป็นเซกเมนต์ความยาว N ตัวอย่าง (เช่น N = 4096) ด้วยการซ้อนทับ (overlap) 50% และทำมาตรฐานข้อมูลแบบ z-score เพื่อลดอิทธิพลจากสเกลของแอมพลิจูด ก่อนแบ่งเป็นชุด train/validation/test โดยยึดการแบ่งตามเงื่อนไขการทดสอบ (เช่น load หรือ run) เพื่อหลีกเลี่ยงข้อมูลรั่วตามข้อเสนอแนะในงาน benchmark [3], [10]

III.II การสร้างภาพสเกโลแกรมด้วย CWT

สำหรับแต่ละเซกเมนต์ คำนวณ CWT ด้วย Wavelet Toolbox และสร้างสเกโลแกรมจากค่าสัมบูรณ์ของค่าสัมประสิทธิ์ CWT [4], [5]. งานนี้ใช้เวฟเลตเชิงวิเคราะห์ (เช่น Morse) และกำหนดจำนวน voices-per-octave เพื่อเพิ่มความละเอียดในย่านความถี่ จากนั้นทำการ normalize และปรับขนาดภาพเป็น 224×224 พิกเซล (grayscale) เพื่อป้อนให้เครือข่าย CNN

III.III แบบจำลองและการฝึกสอน
III.III.I วิธีฐาน

สกัดคุณลักษณะเชิงสถิติและสเปกตรัม เช่น RMS, Kurtosis, Skewness, Crest factor, Spectral entropy และ Bandpower แล้วฝึกสอนตัวจำแนก SVM (RBF kernel) และ RandomForest เพื่อเป็น baseline [1]

III.III.II วิธีเสนอ

ออกแบบโครงข่ายประสาทเทียมแบบคอนโวลูชันที่มีโครงสร้างกะทัดรัด (Lightweight CNN) ประกอบด้วยบล็อกชั้น Convolution, Batch Normalization, ReLU และ Max Pooling จำนวนหลายบล็อก พร้อม Dropout ก่อน Fully Connected สำหรับจำแนก 4 คลาส ฝึกสอนด้วย Adam optimizer และ mini-batch training [11], [12]

เพื่อให้สามารถทำซ้ำผลการทดลองได้อย่างตรวจสอบได้ พารามิเตอร์ที่เกี่ยวข้องกับการฝึกสอนและโครงสร้างแบบจำลองทั้งหมดถูกบันทึกไว้ ได้แก่ ค่าการสุ่มเริ่มต้น โครงสร้างเครือข่าย ค่าพารามิเตอร์การฝึกสอน และแฟ้มแบบจำลองที่ผ่านการฝึกสอนแล้ว

IV. ผลการวิจัย

การประเมินผลการจำแนกความผิดปกติของลูกปืนดำเนินการบนชุดข้อมูล CWRU โดยกำหนดการจำแนกเป็น 4 คลาส ได้แก่ ความเสียหายเม็ดลูกปืน (Ball) ความเสียหายวงแหวนด้านใน (InnerRace) สภาพปกติ (Normal) และความเสียหายวงแหวนด้านนอก (OuterRace) ทั้งนี้ได้เปรียบเทียบ 3 วิธี:

  • (1) วิธีที่นำเสนอ: ภาพสเกโลแกรมจาก CWT ร่วมกับ Lightweight CNN
  • (2) วิธีฐาน SVM: คุณลักษณะเชิงสถิติร่วมกับ Support Vector Machine
  • (3) วิธีฐาน Random Forest: คุณลักษณะเชิงสถิติร่วมกับ Random Forest

IV.IV ผลลัพธ์เชิงคุณภาพของภาพสเกโลแกรม

รูปที่ IV แสดงตัวอย่างภาพสเกโลแกรมจาก CWT สำหรับทั้ง 4 คลาส ซึ่งสะท้อนรูปแบบการกระจายพลังงานของสัญญาณในโดเมนเวลา–ความถี่ที่แตกต่างกันตามชนิดความผิดปกติ และช่วยสนับสนุนว่าแนวทางการแปลงสัญญาณเป็นภาพสเกโลแกรมสามารถทำให้แบบจำลองเรียนรู้ลักษณะจำเพาะของแต่ละคลาสได้อย่างมีประสิทธิภาพ

รูปที่ IV. ตัวอย่างภาพสเกโลแกรมของข้อมูล 4 คลาส (Ball, InnerRace, Normal, OuterRace)

IV.V สมการและวิธีคำนวณตัวชี้วัดที่ใช้ในการประเมินผล

เพื่อให้การประเมินผลมีความชัดเจนและสามารถทำซ้ำได้ งานวิจัยนี้นิยามเมทริกซ์ความสับสน (Confusion Matrix) ของการจำแนกแบบหลายคลาส (จำนวนคลาส $K = 4$) ดังนี้ ให้ $\mathbf{M} \in \mathbb{N}^{K \times K}$ เป็นเมทริกซ์ความสับสนที่มีสมาชิก $M_{ij}$ แทน "จำนวนตัวอย่างที่มีคลาสจริงเป็น $i$ แต่ถูกทำนายเป็น $j$"

เมื่อพิจารณาคลาสที่ $c$ (เช่น Ball, InnerRace, Normal, OuterRace) สามารถคำนวณค่าพื้นฐานสำหรับการวัดผลได้ดังนี้:

(1) $$TP_c = M_{cc}$$

โดยที่ $TP_c$ คือจำนวนที่ทำนายถูกในคลาส $c$

(2) $$FN_c = \sum_{j=1, j \neq c}^{K} M_{cj}$$

โดยที่ $FN_c$ คือจำนวนของคลาสจริง $c$ ที่ทำนายผิดไปเป็นคลาสอื่น

(3) $$FP_c = \sum_{i=1, i \neq c}^{K} M_{ic}$$

โดยที่ $FP_c$ คือจำนวนที่ถูกทำนายเป็นคลาส $c$ แต่คลาสจริงไม่ใช่ $c$

(4) $$TN_c = \sum_{i=1}^{K} \sum_{j=1}^{K} M_{ij} - TP_c - FP_c - FN_c$$

โดยที่ $TN_c$ คือจำนวนที่เหลือทั้งหมด

จากค่าดังกล่าว นิยามตัวชี้วัดหลักที่ใช้รายงานผลในงานวิจัยนี้ ได้แก่:

IV.V.I ความเที่ยง (Precision)
(5) $$Precision_c = \frac{TP_c}{TP_c + FP_c}$$
IV.V.II ความครอบคลุม (Recall)
(6) $$Recall_c = \frac{TP_c}{TP_c + FN_c}$$
IV.V.III ค่าคะแนนเอฟวัน (F1-score)
(7) $$F1_c = \frac{2 \cdot Precision_c \cdot Recall_c}{Precision_c + Recall_c}$$
IV.V.IV ความถูกต้อง (Accuracy)

ให้ $N = \sum_{i=1}^{K} \sum_{j=1}^{K} M_{ij}$ เป็นจำนวนตัวอย่างทั้งหมด และ $\sum_{c=1}^{K} M_{cc}$ เป็นจำนวนตัวอย่างที่ทำนายถูกทั้งหมด จะได้

(8) $$Accuracy = \frac{\sum_{c=1}^{K} M_{cc}}{N}$$

เนื่องจากเป็นการจำแนกแบบหลายคลาส งานวิจัยนี้สรุปผลภาพรวมด้วยค่า Macro-average ซึ่งเป็นค่าเฉลี่ยของตัวชี้วัดในแต่ละคลาส โดยให้น้ำหนักเท่ากันทุกคลาส ดังนี้:

(9) $$Macro\text{-}Precision = \frac{1}{K} \sum_{c=1}^{K} Precision_c$$
(10) $$Macro\text{-}Recall = \frac{1}{K} \sum_{c=1}^{K} Recall_c$$
(11) $$Macro\text{-}F1 = \frac{1}{K} \sum_{c=1}^{K} F1_c$$

สำหรับเมทริกซ์ความสับสนที่แสดงเป็น "ร้อยละ" ในรูปประกอบ สามารถคำนวณได้จากการทำให้เป็นสัดส่วนตามผลรวมรายแถวหรือรายคอลัมน์ ดังนี้:

(12) $$P_{ij}^{(row)} = \frac{M_{ij}}{\sum_{j=1}^{K} M_{ij}} \times 100$$

ร้อยละแบบรายแถว (row-normalized) ซึ่งแสดงสัดส่วนการทำนายในแต่ละคลาสจริง

(13) $$P_{ij}^{(col)} = \frac{M_{ij}}{\sum_{i=1}^{K} M_{ij}} \times 100$$

ร้อยละแบบรายคอลัมน์ (column-normalized) ซึ่งแสดงสัดส่วนความถูกต้องของการทำนายแต่ละคลาส

สมการและตัวชี้วัดทั้งหมดข้างต้นสามารถนำไปคำนวณผลการทดลองของทั้งวิธีที่เสนอ (CNN จากภาพสเกโลแกรม CWT) และวิธีฐาน (SVM, Random Forest) เพื่อให้สามารถเปรียบเทียบผลได้อย่างเป็นธรรมและสอดคล้องกันในทุกโมเดล

V. สรุปและอภิปรายผล

งานวิจัยนี้นำเสนอแนวทาง MATLAB-only สำหรับวินิจฉัยความผิดปกติของลูกปืนจากสัญญาณสั่นสะเทือนด้วยการแปลง CWT เพื่อสร้างภาพสเกโลแกรม และจำแนกด้วย Lightweight CNN โดยออกแบบให้รองรับการจำแนก 4 คลาสบนชุดข้อมูล CWRU พร้อมเปรียบเทียบกับวิธีฐานที่ใช้คุณลักษณะเชิงสถิติร่วมกับ SVM และ RandomForest

การประเมินผลจะรายงานด้วยตัวชี้วัดมาตรฐานและ Confusion Matrix ภายใต้การแบ่งข้อมูลที่ลดความเสี่ยง data leakage ตามงาน benchmark [3], [10]

แนวทางในอนาคต
  • เพิ่มการทดสอบข้ามโหลด/ข้ามความเร็ว (Cross-load/Cross-speed Testing)
  • ปรับให้โมเดลทนต่อสัญญาณรบกวน (Noise Robustness)
  • รองรับสภาพการใช้งานจริงในโรงงาน (Real-world Industrial Deployment)
